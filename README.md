This is My [Next.js](https://nextjs.org/) Protfolio project bootstrapped with [`create-next-app`]

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

:rocket: This Project uses Internationalization with React-intl 

:rocket: Tailwind is used for styling
