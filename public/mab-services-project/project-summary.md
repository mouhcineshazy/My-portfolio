MAB Services — Professional lead-generation website for an insurance broker and financial security advisor licensed in Québec (AMF) and Ontario (FSRA). The site converts visitors into free consultation requests through a
bilingual (French/English) experience optimized for local SEO and AI search engines.

  ---
What it does
- Presents insurance and savings/investment services in French and English
- Captures consultation leads via a contact form with branded email notifications
- Hosts a free masterclass with live Cal.com booking and countdown timer
- Structured data (JSON-LD) and llms.txt for Google and AI citation (ChatGPT, Perplexity, Claude)

  ---
Tech Stack
- Framework — Next.js 15 (App Router, Server Actions, ISR)
- Language — TypeScript (strict)
- Styling — Tailwind CSS v3 + CSS custom properties
- i18n — next-intl v4 (FR/EN bilingual routing)
- Forms — React Hook Form + Zod validation
- Email — Resend (transactional email)
- Booking — Cal.com embed (consultation and masterclass)
- Hosting — Netlify (CI/CD via GitHub Actions)
- SEO/GEO — JSON-LD schema graph, hreflang, sitemap, robots.txt with AI crawler allowlist
